Shortify

A simple to use api for the Goo.gl URL Shortener service.

How to call the API ?

http://shortify.appspot.com/shorty?url=[longURL]
Note : Change the [longURL] with the URL you want to shortify.


Example

Request
http://shortify.appspot.com/shorty?url=http://www.google.com
Response
{"short_url":"http://goo.gl/fbsS","added_to_history":false}
QR Code

Just add the .qr extension in the end of your goo.gl URL. For example, if your Goo.gl url is http://goo.gl/fbsS. Then add .qr extension at the end of your url. So your QR Code URL will be http://goo.gl/fbsS.qr.

Error Handling

If no URL is specified -
{"error_message":"Bad request"}
